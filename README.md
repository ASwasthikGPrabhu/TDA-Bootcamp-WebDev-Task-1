# ShopEasy - Modern E-Commerce Landing Page

A modern and responsive e-commerce landing page built using React, Vite, Tailwind CSS, and GSAP animations.

## Live Demo

https://tda-bootcamp-web-dev-task-1.vercel.app

## Features

* Responsive design for desktop, tablet, and mobile devices
* Dark violet premium UI theme
* Animated splash screen
* Mobile hamburger navigation
* Interactive product cards with hover effects
* Smooth GSAP animations
* Responsive footer with social media icons
* Modern newsletter subscription section
* Flash deals section
* Customer reviews section
* Category browsing section

## Tech Stack

* React
* Vite
* Tailwind CSS
* GSAP
* React Icons
